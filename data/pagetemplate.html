i wrote a subsurface xml database parser in go as a fun way of exploring how the standard library parser works, and then decided to build a web server for fetching the dive data and the hypermedia interface for browsing.

this site is intended for personal use, but the implementation is open source and can be found by following the link in the header. i tested the parser and designed the data structures based on my dive log only, there's plenty of room for fixes and extensions.

if you get interested in using this, and encounter any bugs, please open an issue on github.

some ideas that i intend to work on: fetch the database directly from subsurface servers if possible, display depth and temperature profiles on dive pages, add a historical view / stats page for aggregation functions etc.
